We need fonts

We are unique. We show who we are through typography. We want our identities to be seen and heard through fonts.Today, billions of people need fonts that are custom designed for themselves and their businesses.

Metapolator enables all font lovers, from children to professional type designer, to create unique typefaces within large font families.

Tools

Workflow

Standards

Type designers need a toolbox to build a single typeface out into a family of styles. Metapolator provides an initial set of tools for common processes, but it is fully extendable.

  • spacing (tighter or looser)
  • weight (regular to bold, or to light)
  • width (normal to narrow, or extended)
  • x-height (increased or decreased, without distortions)

Type designers need tools that snap into their existing workflow.

The text-based and interchangeable UFO source format has become an industry standard. Metapolator imports and exports UFO fonts, to add a new link in the chain of font style and family creation. As a web app it is built with an industry standard too: AngularJS.

Lets start using one single font. As soon we have more than one master we can create childrens and siblings. The replication is not limited to latin alphabets. It is possible to create a hindi font out of a latin style by translating parameters from one source to another.
