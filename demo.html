Where is the demo?

Read this page to get an idea what this is all about or get the lates build from https://github.com/metapolator/metapolator

Can we extrapolate from a single master?

Yes, Metapolator is calculating a skeleton from the glyph outline. From the skeleton to the original outline or to a parametrised version, we can create new inter- and extrapolations.

Why do I need to tag points?

Metapolator wants to know how and where you want change an outline. For this we have to 'name' a few parts of a glyph. Only if a part is known for the system it can be controlled.

When do I need to tag points?

Glyphs gets grouped into its type when loading a UFO. A capital letter is therefore in the group 'capitals' and by changing values on vertical metrics we can adjust the height of the capital letter. If we want to specify single points we click on a point in the editor and assign it to a group. In the case of lowercase 'b' we want to tag the ascender stroke as 'b' gets into the group x-height by default (tagging the ascender is less work than tag all x-height related points).

Can I design a glyph from scratch?

Yes, but do it on paper! :) Metapolator is a system to create new styles and font families from existing fonts outlines. The system is designed to work seamlessly within the working process of type design. The design process could be for example; sketching on paper -- creating outlines in a font editor -- making new styles and families in Metapolator -- refine ufo’s in a font editor.

Is Metapolator using metaphysics?

No, the term ‘meta’ refers to Donald Knuth’s Metafont, a programming language for digital fonts. Mr. Knuth is one of the founders of contemporary mathematics and computer programming. There is no "magic" going on in Metafont, only pure, clean and perfect calculations.

Why is the slider stack always visible?

A core element is the metapolation settings. To compare values with other instances the controller is on the same view as the instances itself. Controllers can be hidden individually or by pressing ‘h’.

Can I use my renaissance antiqua in Metapolator?

Yes, for interpolation using at least 2 masters no extra work is needed. Metapolation from a single master requires tagging consistently using pairs of points.

I dont want to wait until the new demo is ready, how can I test Metapolator anyway?

Use our new command line tool, which does the same interpolation calculation as the interface.

Why is Metapolator dealing with control points and pen points?

The level of abstraction is up to the user. You can have total control of all anchor points and control points or leave the point calculation to Metapolator. You decide how much of 'automatic' design you want.

When I want to apply some metafont function to the tagged points, how do I do it? How does the serif function work? Eg, make all cap-serif points 2x taller?

How does the serif function work?

Serifs can be ‘attached’ to every corner point of a stroke. The user gets control of the serif by setting parameters like serif height, slab, width and darkness.

Can I make an italic font from a regular cut?

Yes, you can use the parameter ‘slant’ which is using optical correction when slanting outlines and substitute common (real) italics glyphs like g, a, or y (italic font)

What is metapolation?

Interpolation enhanced with metafont parameters, for example to change the x-height of a font we use some metafont functions to calculate the new curve.

What is a card?

A card is a specimen which shows the instance from one or muliple used axes.

Can I change the specimen?

We can choose from glyph, paragraph or waterfall.

Can I compare different styles within one card?

Yes, you can copy paste text from another card and paste it into your preferred card. The style and metapolation settings also gets copied.

Where's the preview for each master?

You can select every master that has been loaded into Metapolator. Click on checkbox "display master". This brings the master into the first card and moves all previous cards one card to the right.

How can I edit a master?

You can use the master editor to tag points or move their position. To make big changes on the outline, you can save your project, open the UFO in your favorite font editor and load it back to metapolator.

Can I change the layout of the cards?

Yes, use the column settings to change from 1, 2, or 4 column layout

I want to use waterfall instead of glyph preview.

The user can choose between glyph view, waterfall or paragraph.

I’m using my font styles created with metapolator on my website, if I change one of the styles in metapolator, will it change on my website too?

You can use the direct webfont link to embed an instance into your website. This allows to make changes on the fonts which affects the design of your page.

How many axes and masters can I create?

Its modular design allows the user to create unlimited amount of axes and link unlimited masters to create very complex interpolation environments. The environment is not limited to ‘multidimensions’ as we believe a dimensional space limits the users imagination and makes it complicated. Its still possible though to recreate such a model.

Can I create new masters within Metapolator?

Yes, a user can create a new master from every instance or duplicate the master itself.

How can I display the master and not the instance?

The cards only shows instances. If we want to show a master, we can create a new instance and name and set all slider values to its master.

I’m lost with all these masters and instances I’ve created.

You can use the tree diagram to get an overview for all created masters and instances within your project.

I don’t like the logo, can I change it?

Yes, click on the logo and you can set your own font and the recursion value.

Can I run Metapolator on my computer?

Metapolator is built as a web application. Its possible to install everything on a local machine using a local server setup. The installation process is described on git: https://github.com/metapolator/metapolator
